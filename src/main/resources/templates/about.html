About / Contribute

Howdy, I'm Ben, welcome to chicken-api 🐔🌐 I wanted to make a fun project to host everything chicken. I want this to grow into the de facto standard for chicken-related data. If you have a chance please add a breed or your own chicken! Contributions are welcome to the code too, feel free to submit a pull request!

How to Submit Data

To add a new chicken breed, please make a post request to /api/v1/breeds. We will review and add to the production database. To add new features or requests, please open an issue on our GitHub repository.
