Welcome to Chicken API

A simple API for all your chicken-related needs.

About the API

  • Built with Spring Boot and Kotlin
  • RESTful endpoints for chicken data
  • OpenAPI (Swagger) documentation available
  • Integrated with Google Sheets for data storage

How to Use

  • Explore endpoints and try requests in the OpenAPI Docs
  • Check the /actuator/health endpoint for health status
  • See the /api/chickens endpoint for chicken data
